O QUE O PROGRAMA BAIXEASSISTA FAZ.
==========================
- Remove o limite de tempo imposto pelo servidores de vídeos online.
- Baixe e exibe, no player interno ou externo, o arquivo de vídeo.
- Exibe o arquivo de vídeo, direto do programa.
- Resume downloads parados anteriormente.
- Gera histórico das urls de vídeos baixados.
- Acelera o download, pois diversas conexões podem ser criadas ao mesmo tempo.
- e muito mais...

PLAYER DE VÍDEO
==========================
- O programa BaixeAssista, oferece duas opções de player: 
-- O embutido(jwplayer - usado largamente na internet).
-- E o externo(como splayer; FLV player e alguns outros), deve ser baixado e instalado separadamente.

NAVEGADOR
==========================
- Server de base para programa. Ele é a fonte de urls.
- Com o navegador, é possível extrair as urs dos vídeos, seja por "thumbnail" ou captura automática.
- Com um simples click no "thumbnail", a url é captura e inserida do controle correto.
- Clicando no botão "Baixar", o download começa imediatamente.
** Dica! - É possível baixar vídeos sem usar o navegador. Basta copiar e colar a url.
** Dica! - É importante saber se o programa suporta a url fornecida para ele, veja a lista baixo.

SERVIDORES PROXIES
==========================
- Como o objetivo é tornar o programa anônimo para o mundo da internet, em realação a 
transferência dos vídeos, vários ips diferentes são usados.
- Para tornar mais fácil o uso de servidores proxies, um pesquisador de ips está disponível.
- Uma vez configurado apropriadamente, ele cumpre seu trabalho e gera uma nova lista de ips,
com servidores ativos e prontos para transferir arquivos de vídeo.
